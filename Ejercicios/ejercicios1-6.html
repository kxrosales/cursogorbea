Introduccion

SQL(Lneguaje de consulta estructurado) es un lenguaje de programacion para trabajar con la base de datos relacionales como MySQL, Oracle, etc.SQL es un acrónimo en inglés para Structured Query Language. Un Lenguaje de Consulta Estructurado. Un tipo de lenguaje de programación que te permite manipular y descargar datos de una base de datos. Tiene capacidad de hacer cálculos avanzados y álgebra. Es utilizado en la mayoría de empresas que almacenan datos en una base de datos. Ha sido y sigue siendo el lenguaje de programación más usado para bases de datos relacionales.SQL es un acrónimo en inglés para Structured Query Language. Un Lenguaje de Consulta Estructurado.
